# Sample Case Documents

This folder contains generated test documents for the demo PangoChain cases.

Run this from the repository root to regenerate the `.txt` and `.pdf` files:

```powershell
node scripts/generate-sample-case-documents.mjs
```

Use the `.txt` files when you want easy AI assistant/RAG indexing input. Use the matching `.pdf` files when you want to test upload, document metadata, vault listing, and case attachment flows.
